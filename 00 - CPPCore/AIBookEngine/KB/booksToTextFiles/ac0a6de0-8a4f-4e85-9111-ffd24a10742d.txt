Compact Key for Schools is a focused, 50 - 60 hour course for the Cambridge English: Key (KET) for Schools exam. This Student's Book features twelve topic-based units with focused exam preparation to maximise the performance of school-age learners. Units contain pages on Reading, Writing, Listening, Speaking, Grammar and Vocabulary. A Grammar reference section covers key areas in the syllabus and unit-based wordlists include target vocabulary with definitions. A Speaking Guide provides extra support for this paper. The accompanying CD-ROM provides interactive grammar, vocabulary and exam skills tasks including listening. A Testbank access code for four online Cambridge English: Key for Schools practice tests is included with this Student's Book (class code from teacher required).
