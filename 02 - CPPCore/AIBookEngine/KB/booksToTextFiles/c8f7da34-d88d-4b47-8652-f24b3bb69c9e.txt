ORIENTACIÓN A OBJETOS CON JAVA Y UML - 2DA. EDICIÓN - 978-987-1104-88-8Edad recomendada: Adultos.SINOPSIS¿Es este otro libro de Java v UML? Sí y no. Sí, porque encara los aspectos fundamentales de estas tecnologías. Pero no se queda sólo en ello, sino que brinda las herramientas necesarias para hacerlo pensando en objetos, es decir, utilizando el paradigma que ha demostrado su vigencia y actualidad. El libro llega también a aspectos de programación avanzada, analizando persistencia, concurrencia, y computación distribuida.Fontela encara estos temas con claridad, y teniendo en cuenta a los profesionales que vienen de desarrollar software siguiendo el modelo estructurado.El CD que acompaña al libro contiene todos los programas fuentes del mismo, varios apéndices y el SDK de Java listo para instalar en Linux y Windows, además de algunas herramientas útiles para trabajar con JAVA y UML.
