The ActiveTeach software for Interactive Whiteboards will really help bring  classes alive.    This interactive version of the class materials will allow you to:    focus students' attention on the task in hand  clarify instructions and the mechanics of activities quickly and efficiently  complete exercises and check answers in a fun and motivating way  access all the multimedia resources with a few simple clicks  select and print worksheets from the Teacher's Resource File  plan work and keep track of individual students' progress.  This powerful and flexible tool provides everything needed for the fully  digital classroom.
