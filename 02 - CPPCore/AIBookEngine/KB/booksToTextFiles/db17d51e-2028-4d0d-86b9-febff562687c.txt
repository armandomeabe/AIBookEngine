This second edition updates a course which has proven to be a perfect fit  for classes the world over. This Combo Testmaker CD-ROM and Audio CD  contains the testing material for the 3A and 3B Combos together. All unit  references correspond to the Combo edition.    It gives teachers the flexibility to create and edit their own tests. It is  easy to use while also offering a range of functionality.    Using the CD-ROM, teachers can choose from unit tests and unit pair tests.  'Standard' and 'extra challenge' level options are included. The Testmaker  can produce two versions of each test to prevent students from sharing  answers.    The Audio CD contains all the audio required for test listening components.
