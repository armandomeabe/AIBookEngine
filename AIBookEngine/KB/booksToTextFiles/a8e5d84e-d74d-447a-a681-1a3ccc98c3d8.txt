Un compendio de aperitivos que ofrece recetas fáciles y sencillas de preparar. Incluye consejos para seleccionar los mejores ingredientes. También ofrece trucos para arreglar las cosas cuando estas se tuercen en el último momento. Aperitivos para cada ocasión: elegantes canapés para servir un cóctel, divertidas porciones de pizza para fiestas infantiles, entrantes sofisticados para antes de la cena o bocados saludables bajos en calorías para las veladas nocturnas. Una enorme variedad de ideas para preparar aperitivos: crudités, crostini, brochetas picantes, bocados dulces.
