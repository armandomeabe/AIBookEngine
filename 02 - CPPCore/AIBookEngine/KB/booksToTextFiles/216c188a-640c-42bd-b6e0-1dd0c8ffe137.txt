KET Testbuilder with Answer Key & Audio CD    This book contains four complete practice tests for the KET exam. Ideal for  using in the classroom or as a self-study book, these practice tests will  help students develop the confidence and language skills they need to take  the Key English Test.    Key features:    - With and Without Key editions available   - The With Key edition contains a complete answer key with helpful,  easy-to-use explanations as to why an answer is correct and others are wrong    - The accompanying Audio CD in the With Key edition contains all four  Listening Tests.
